# Git commands for Lab 3

```bash
git checkout master
git pull
git checkout -b lab3

git add .
git commit -m "Add lab 3 Entity Framework implementation"
git push -u origin lab3
```

Якщо PR з `lab2` уже закритий, створити Pull Request з `lab3` у `master`.
Якщо PR з `lab2` ще не закритий, створити Pull Request з `lab3` у `lab2`.
