# Laboratory Work 1 - ClothingStore

Theme: clothing store model.

Projects:
- `ClothingStore.Common` - models, abstractions, extensions and generic CRUD service.
- `ClothingStore.Console` - console application that demonstrates CRUD operations, methods, constructors, static members, delegates, events, extension methods, saving and loading data.

Run:
```bash
dotnet restore
dotnet run --project src/ClothingStore.Console/ClothingStore.Console.csproj
```
